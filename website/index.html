What is AutoLang?

      
AutoLang is a domain-specific language (DSL) for robotics. It turns
         simple commands like MOVE FORWARD 10 into standard C++
         that runs on desktops or embedded targets such as ESP32 and STM32.

    


    

      
Program Structure

      
Every program is wrapped in START and END.
         One command per line. Keywords are case-insensitive.

      
START
  <instructions>
END
